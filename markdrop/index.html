markdrop

Browser-based conversion of documents to Markdown using JavaScript libraries loaded from CDN on first visit. Drop one or more files onto the page and the Markdown appears immediately in result cards with per-file stats, copy, and save. Supports CSV, DOCX, HTML, JSON, PDF, plain text, PPTX, XLSX, and XML. When working with multiple files, use the toolbar to sort, filter, collapse, and select results for batch download as a ZIP archive. No installation, no server, no account.
