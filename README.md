# doba-isbmds

Tools, scripts, and utilities developed during doctoral studies in the **Innovation and Sustainable Business Management in Digital Society** programme at **DOBA University of Applied Sciences**, Maribor.

The programme spans three years and covers research methodology, strategic management and innovation, technological and economic sustainability challenges, and the doctoral dissertation. It combines rigorous academic research with practical application across a range of business and technology domains.

This repository collects practical work produced across courses, seminars, and assignments throughout the programme. It serves as a working toolkit built incrementally alongside the studies, covering areas such as data collection, statistical analysis, document processing, and automation.

Tools vary in language and form depending on the task at hand: Python scripts, R analysis, Bash utilities, data pipelines, visualizations, and more. The choice of language follows the problem rather than a fixed stack. Each tool is built to solve a concrete problem encountered during the programme, kept small and focused, and documented so it remains usable long after the course that prompted it. New tools are added as the programme progresses.

## Tools

| Tool | Language | Description |
| --- | --- | --- |
| [markdrop](markdrop/) | HTML/CSS/JS | Browser-based document to Markdown converter. Drop a file and copy the result. Supports PPTX, DOCX, PDF, XLSX, HTML, CSV, JSON, XML, and plain text. No installation or server required. |
| [md2pptx](md2pptx/) | HTML/CSS/JS | Browser-based Markdown to PowerPoint converter. Write or load a Markdown file, preview slides live, and export as PPTX. No installation or server required. |
| [SlideNotesTTS](SlideNotesTTS/) | Python | Converts presentation Markdown files produced by markitdown into MP3 audio using Microsoft Edge TTS. Supports slide text, speaker notes, or both, with interactive voice selection and parallel generation. |
