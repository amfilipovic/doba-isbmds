OpenAlexSLR

A query builder for systematic literature reviews, drawing on the OpenAlex open catalogue of over 317 million academic works. Compose a Boolean search from multiple concept groups using AND, OR, and NOT connectors, refine by year and publication type, and export any selection to CSV or RIS for your reference manager. Filter and select from loaded results before exporting, or load additional pages as needed. Queries can be saved as JSON files and reloaded across sessions for reproducible searches. Nothing to install, no account or API key needed.
